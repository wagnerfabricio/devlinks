# DevLinks

DevLinks is a responsive link aggregator with theme switching that you can use as a business card on your social networks.

This is one of the practical challenges from the Next program, part of the content available to Rocketseat students.

## About

This is a [Next.js](https://nextjs.org) project with [Tailwind CSS](https://tailwindcss.com) and [Radix UI](https://www.radix-ui.com) components.

The home page is a headless page where all components come from [Prismic](https://prismic.io).

## Getting Started

### Installation

Install all dependencies:

```bash
npm install
```

### Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Prismic Slice Machine

Run the Prismic Slice Machine:

```bash
npm run slicemachine
```
